# line-harness-oss

## 概要

LINE公式アカウントの運用を支える、オープンソースのCRM（顧客関係管理）アプリケーションです。ステップ配信、ブロードキャスト、フォーム、リッチメニュー、スコアリング、自動化など、配信と顧客対応に必要な機能をまとめて扱うためのプロジェクトです。

## 何をするプロジェクトか

- LINE公式アカウント運用の基本機能を一つにまとめる
- 有料ツールに依存せず、自分で拡張できる運用基盤を作る
- 配信、フォーム、リッチメニュー、顧客スコアを同じ文脈で管理する

## 想定ユーザー

LINEで顧客接点を持つ個人・小規模事業者・開発者向けです。
